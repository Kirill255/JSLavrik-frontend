Реализовать файл es6/lesson2/hw.js.

Код, тестирующий модуль есть в файле es6/lesson2.js

1. Простое ДЗ.

	Реализовать класс EmailParser по следующим правилам:
		- в объекте существует только одно реальное поле email
		- также присутствует возможность обратиться к полям
			- isCorrect: (boolean) корректный ли адрес
			- name: (string) часть до собаки || null if !isCorrect
			- domain: (string) часть после собаки || null if !isCorrect

2. Жёсткое ДЗ.

	Реализовать фукнцию watchObj принимающую объект DOM и возвращающую Proxy.
  Которая работает по следующему приниципу: тестируем мы её на элементе DOM, создаётся элемент div и теперь при задании для этого div каких-либо свойств или методов, мы могли бы перехватить этот миг, хотим как-то на это подписаться и как-то отреагировать, в нашем случае просто логировать.
  Например мы закидываем в нашу функцию watchObj наш div и функцию прослушки, функция прослушки выводит на экран какое поле мы обновляем, какое значение туда задаём.
	Подробности смотрите в файле es6/lesson2.js и в конце видео js-2-2.


Данное ДЗ присылайте в виде ровно одного js-файла, содержащего модуль.
Файл назван вашим именем или превдонимом.

Если почтовый клиент будет блокировать отправку js-файла, просто поменяйте расширение на txt.
